# auto_ubuntu

Автоматическая установка и настройка Ubuntu под конкретный сетап — без
интерактивных инсталляторов, без ручной возни с разделами, без «нажми далее».

Суть проста: запускаешь VM с кастомным ISO — через несколько минут получаешь
готовую систему с нужными пакетами, правильной разбивкой диска и настроенным
SSH. Можно пересобрать с нуля хоть каждый день.

---

Проект не претендует на универсальность. Здесь нет поддержки разных
дистрибутивов, нет плагинов, нет конфигов на все случаи жизни. Это
opinionated-набор решений под конкретные задачи и железо — btrfs с подтомами,
i3 как WM, проброс физического диска через QEMU.

Если тебе нужно то же самое — добро пожаловать. Если нужно что-то другое —
форкай и переделывай под себя.
